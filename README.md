## About eduvia

**eduvia** is a modern, all-in-one campus management platform designed to simplify academic life. Built for students and faculty at IHRD College of Engineering Kallooppara, eduvia brings together schedules, notes, events, and intelligent study assistance into a single, beautiful interface.

> *Your academic journey, simplified.*

---

## Features

### For Students

| Feature | Description |
|---------|-------------|
| **Smart Timetable** | View your complete semester schedule with color-coded classes, room locations, and faculty details |
| **Course Notes** | Access PDF study materials organized by semester, branch, and subject — all in one place |
| **Classroom Finder** | Never get lost! Quickly locate any classroom or lab on campus |
| **Events Hub** | Stay updated on seminars, workshops, cultural events, and important deadlines |
| **eduvia AI** | Ask questions about your studies, get explanations, and receive personalized academic guidance |
| **Lost & Found** | Report lost items or check if someone found what you're looking for |
| **Personal Profile** | Manage your academic profile, semester info, and contact details |

### For Faculty & Administrators

| Feature | Description |
|---------|-------------|
| **Notes Management** | Upload and organize course materials with Google Drive integration |
| **Timetable Editor** | Create and modify class schedules for different branches and semesters |
| **Event Creation** | Publish campus-wide announcements and events |
| **Lost & Found Admin** | Manage reported items and mark them as claimed |
| **Faculty Requests** | *(Super Admin)* Review and approve faculty access requests |

## Getting Started

### For Users

1. **Visit eduvia** at your institution's portal
2. Click **"Sign in with Google"** 
3. Use your **institutional email** to authenticate
4. Complete your profile with semester and branch details
5. Start exploring!

### Quick Tips

- Check your **timetable** first thing every semester
- **Download notes** for offline access
- Keep an eye on **Events** for important dates
- Use **eduvia AI** for quick study help

## Security

This project takes security seriously:

- **Environment-based secrets** — No hardcoded credentials
- **Row Level Security (RLS)** — Database-level access control
- **Role-based access** — Student, Admin, and Super Admin tiers
- **OAuth 2.0** — Secure Google authentication
- **Server-side validation** — All sensitive operations validated on backend
- **HTTPS only** — Encrypted data transmission
